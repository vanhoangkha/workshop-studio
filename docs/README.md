# Documentation

This directory contains comprehensive documentation for AWS Workshop Studio development and sample code publishing.

## Available Documentation

### Workshop Studio Development

- **[AWS Workshop Studio Development Guide](AWS_WORKSHOP_STUDIO_DEVELOPMENT_GUIDE.md)** - Complete guide for creating workshops in AWS Workshop Studio, covering the entire development workflow from initial setup to publication.

- **[AWS Workshop Studio Development Guide Part 2](AWS_WORKSHOP_STUDIO_DEVELOPMENT_GUIDE_PART2.md)** - Advanced topics including infrastructure setup, testing, and deployment strategies.

- **[AWS Workshop Studio Development Guide Part 3](AWS_WORKSHOP_STUDIO_DEVELOPMENT_GUIDE_PART3.md)** - Best practices, troubleshooting, and optimization techniques for workshop development.

- **[AWS Workshop Studio Guideline](AWS_WORKSHOP_STUDIO_GUIDELINE.md)** - Official guidelines and standards for creating high-quality workshops that meet AWS requirements.

### Migration Guides

- **[Hugo to Workshop Studio Migration](HUGO_TO_WORKSHOP_STUDIO_MIGRATION.md)** - Guide for migrating existing Hugo-based workshops to AWS Workshop Studio format.

- **[Migration Complete Guide](MIGRATION_COMPLETE_GUIDE.md)** - Step-by-step instructions for complete workshop migration, including content conversion and testing.

## Quick Links

### Getting Started
1. Start with the [AWS Workshop Studio Development Guide](AWS_WORKSHOP_STUDIO_DEVELOPMENT_GUIDE.md)
2. Review the [AWS Workshop Studio Guideline](AWS_WORKSHOP_STUDIO_GUIDELINE.md)
3. Explore example workshops in the `examples/` directory

### Migration
1. Read the [Hugo to Workshop Studio Migration](HUGO_TO_WORKSHOP_STUDIO_MIGRATION.md) guide
2. Follow the [Migration Complete Guide](MIGRATION_COMPLETE_GUIDE.md)
3. Use the migration script in the root directory
